Тест-план
1. Цель исследования
Необходимо провести Исследовательское Тестирование сайта https://ftb.mentorpiece.org/ (проверить для RU версии), используя методику туров
2. Объект тестирования
Веб-сайт https://ftb.mentorpiece.org/ и его ключевые функциональные зоны (главная, каталог, корзина, оформление заказа, личный кабинет).
3. Инструменты
Браузеры: Chrome, Yandex
Баг-трекер Github.com
4. Подготовка и Сессия
Время: 30-60 минут на один тур
Метод: Пройти по району сайта (функционалу), фиксируя аномалии.
5. Примерное расписание туров
1	«Деловой центр»	Пройти основные сценарии: регистрация -> поиск товара -> добавление в корзину -> оформление заказа. Ввод валидных данных.
2	«Исторический район»	Проверка функционала, навигации по ссылкам, работоспособности страниц и ссылок.
3	«Район развлечений»	Тестирование интерактивных элементов: слайдеры, видео, анимации, Проверка на зависания.
4	«Неблагополучный район»	Попытка сломать сайт: ввод спецсимволов, пустые поля, некорректные форматы данных
5	«Туристический район»	Изучение сайта «новичком»: удобство навигации, понятность интерфейса, читаемость контента.
6	«Район отелей» (Интеграции)	Проверка взаимодействия с внешними системами: оплата, отправка email, авторизация через соцсети.
6. Критерии завершения
Все запланированные туры выполнены.
Найденные дефекты задокументированы.
Оценены риски и функциональная целостность.
7. Результаты
Отчет о сессиях (какие туры пройдены, затраченное время).
Список найденных багов (баг-репорты).
Заметки по улучшению юзабилити.
